# Trafik

Projet React visant à proposer aux utilisateurs une recherche d'itinéraires allant d'un point A à un point B, pour une ligne donnée.

## Partie Front

### Pré-requis

```bash
npm install

npm start
```

### Url du site en prod

`https://trafik.netlify.com`
